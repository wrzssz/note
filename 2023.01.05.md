# Kali开机root自动登录
1. 编辑`/etc/lightdm/lightdm.conf`，修改以下行
    ```
    [Seat:*]
    autologin-user=root
    autologin-user-timeout=0
    ```
2. 编辑`/etc/pam.d/lightdm-autologin`文件，注释该行
    ```
    # auth required pam_succeed_if.so user != root quiet_success
    ```
3. 运行`xfce4-power-manager-settings`，设置为不锁屏
