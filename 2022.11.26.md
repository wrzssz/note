# 安装完Ubuntu 22.04之后
* 更新和源
    ```bash
    cp -f /etc/apt/sources.list{,.bak}
    echo 'deb https://mirrors.ustc.edu.cn/ubuntu/ jammy main restricted universe multiverse
    deb https://mirrors.ustc.edu.cn/ubuntu/ jammy-updates main restricted universe multiverse
    deb https://mirrors.ustc.edu.cn/ubuntu/ jammy-backports main restricted universe multiverse
    deb https://mirrors.ustc.edu.cn/ubuntu jammy-security main restricted universe multiverse' > /etc/apt/sources.list

    apt update -y
    apt dist-upgrade -y
    apt autoremove -y
    ```
* 启用root并删除普通用户
    ```bash
    sudo -i
    passwd
    # chpasswd <<< 'root:iampassword'
    loginctl list-sessions
    loginctl kill-session 15
    userdel -r 'normaluser'
    ```
* 配置sshd
    ```bash
    apt install openssh-server -y
    # echo 'PermitRootLogin yes
    # PasswordAuthentication yes' >> /etc/ssh/sshd_config
    # ssh-keygen -f /root/.ssh/id_rsa -N '' -q
    # cat /root/.ssh/id_rsa.pub > /root/.ssh/authorized_keys
    mkdir /root/.ssh
    echo 'ssh-rsa AAAAB......YYYYZ' > /root/.ssh/authorized_keys
    systemctl enable openvpn.service --now
    ```
